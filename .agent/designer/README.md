## 이 저장소의 디자인 원칙

- dark editorial 톤 유지
- teal accent는 강조 지점에만 사용
- 정보량이 많아도 읽기 흐름이 무너지지 않게 구성
- 작은 인터랙션과 얇은 border 위주로 안정감을 유지
