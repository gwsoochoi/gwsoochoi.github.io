# Shared

> 역할 공통으로 참고하는 프로젝트 문서 모음.

---

## 포함 문서

| 경로 | 용도 |
|------|------|
| `project-overview.md` | 저장소 구조, 제약, 변경 hotspot 파악 |
| `briefs/` | 중간 이상 크기의 변경 요청 정리 |
| `decisions/` | 구조적 결정 기록 |

---

## 언제 쓰나

- 작업 시작 전에 현재 저장소 구조를 빠르게 파악할 때
- 변경 요구사항이 여러 파일에 걸쳐 있어 brief가 필요할 때
- 나중에 다시 설명해야 하는 구조적 선택을 기록할 때
